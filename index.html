Machine & Deep Learning Projects

Generalized Machine Learning Predictor (GMLP)

This ML model is built to be able to train and make predictions based on any uploaded .csv or excel file. The user needs not worry about data pre-processing and other requirements prior to the training.
Upon uploading the file, the user needs to add the property (target) to be predicted.
The training is carried out with the following models

  • Support Vector Classifier
  • Linear SVC
  • Logistic Regression
  • Decision Tree Classifier
  • Random Forest Classifier

To see how the model works, click here. The source codes for the web app are available here on GitHub. For easy check out the Jupyter Notebook version of the source code

Text & Document Classification (TDC)

This is a DL model trained to summarize and provide the wordcloud image of any lengthy document (such as blogs, articles, etc ).
This gives the reader an idea of the document before deciding whether to read the whole document. The length of the summary is provided by the user (default value is 30%).
I am currently working on hosting this application, but the source codes of how the model works for web hosting are available here.
The Jupyter Notebook version of the same source codes is found here.
