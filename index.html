Machine & Deep Learning Projects

Generalized Machine Learning Predictor (GMLP)

This ML model is built to be able to train and make prediction based on any uploaded .csv or excel file. The user needs not worry about data pre-processing and other requirements prior to the training.
Upon uploading the file, the user needs to add the property (target) to be predicted.
The training is carried out with the following models - Support Vector Classifier, Linear SVC, Logistic Regression, Decision Tree Classifier, Random Forest Classifier. Click here to see how the model works. To The source code available of GitHub
